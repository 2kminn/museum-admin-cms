# ArtAR Busan · Admin CMS

React + TypeScript + Tailwind CSS 기반의 관리자 대시보드 템플릿입니다.

## 실행

```bash
npm install
npm run dev
```

## 포함 기능

- 좌측 사이드바 메뉴: 행사 관리 / 장소 및 작품 설정 / 데이터 통계 / 테마 편집기
- 상단 행사 선택(Event Selector) 드롭다운
- 모바일 대응: 햄버거 메뉴 + 오버레이 드로어
- PWA 기본 구성: `public/manifest.webmanifest`, `public/sw.js`
